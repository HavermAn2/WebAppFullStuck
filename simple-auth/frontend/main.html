Simple Auth Web App

About the Project

This web application is designed to demonstrate simple and secure authentication mechanisms using Node.js, Express, and a modern frontend interface. It's a great starting point for developers who want to understand how to build secure login systems with route protection and user sessions.

Features

  • User registration and login
  • Secure password storage
  • Session-based authentication
  • Frontend-backend separation
  • Scalable architecture
  • Custom UI with animated sidebar

Getting Started

Clone the repository, install dependencies using npm install, and run the project using node server.js. You can start exploring the features and customizing it to your needs.
